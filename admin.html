Statistiques du site

Configuration Google Analytics

Entre ton ID de propriété GA4 (un nombre, ex: 123456789) et ton Client ID OAuth2 pour afficher les statistiques directement ici.

Comment obtenir ces informations ?
ID de propriété GA4 :
1. Va sur analytics.google.com
2. En haut à gauche clique sur la propriété → l'ID est affiché (un nombre)
3. Ou : Admin → Paramètres de la propriété → ID de propriété

Client ID OAuth2 :
1. console.cloud.google.com → Nouveau projet → Activer Google Analytics Data API
2. API et services → Identifiants → Créer → ID client OAuth 2.0 → Application Web
3. Origines JS autorisées : ajoute https://bdecreadien.fr et https://bdecreadien.pages.dev
4. Copie l'ID client (se termine par .apps.googleusercontent.com)
Google Analytics complet Cloudflare Analytics

Notifications push

Les notifications push permettent d'alerter les étudiants des nouveaux événements directement sur leur téléphone ou navigateur.

Pour activer :
1. Crée un compte sur onesignal.com
2. New App → Web → entre bdecreadien.fr
3. Copie ton App ID et remplace YOUR_ONESIGNAL_APP_ID dans les fichiers HTML du site
4. Décommente le bloc OneSignal dans chaque page HTML publique
